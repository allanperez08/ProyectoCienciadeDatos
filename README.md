# Proyecto Final: Modelo entrenado para predecir Simulador de Préstamos
Este proyecto utiliza un modelo de Machine Learning (LGBM) para predecir el riesgo de incumplimiento de un préstamo, entrenado con un dataset de 2.2 millones de filas.


### 1. Descargar el Dataset en el siguiente link
https://drive.google.com/file/d/11ikQ0UTjBROZ0spzhmpxjREERp-UhSD-/view?usp=sharing

**IMPORTANTE:** Coloca el archivo `loan.csv` en esta misma carpeta (`Proyecto_Prestamos/`).


### 2. Instalar las Dependencias
En tu terminal, ejecuta:

pip install -r requirements.txt

### 3. Ejecutar la aplicacion
streamlit run app.py
